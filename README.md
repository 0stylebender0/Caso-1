# DUA Streamliner

---
## Problema
La elaboración del Documento Único Aduanero (DUA) es un proceso manual, repetitivo y propenso a errores, 
que requiere interpretar múltiples documentos fuente en formatos heterogéneos como Excel, Word, PDF e imágenes escaneadas. 
Este trabajo depende fuertemente del conocimiento experto de agentes aduaneros, generando cuellos de botella, 
costos operativos elevados y riesgo de rechazos o multas ante la autoridad aduanera. 
Este proyecto busca automatizar este proceso mediante inteligencia artificial, 
extrayendo y mapeando automáticamente la información relevante al template oficial del Ministerio de Hacienda de Costa Rica.
